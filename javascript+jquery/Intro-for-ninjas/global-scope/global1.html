This file examines the global object created.

Please check out the attached javascript file. You will find that it is empty. However it was successfully executed and as a consequence the global object has been created in memory for you.

Open up your console and try the following:

  1. try to output the value of the variable "this" by writing "this" in the console and hitting enter.
  2. Try to output the "window" object by writing "window" and hitting enter.

In most browsers the global object is the "window" object. When Javascript is initiated the "this" variable referes to the window object.
